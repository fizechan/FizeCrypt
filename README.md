# FizeCrypt
各式各样的编码解码类库。

FizeCrypt 是对各式各样的编码解码类库。
FizeCrypt 是 Fize 项目族群的底层依赖，众多 Fize 项目依赖 FizeCrypt ，同时欢迎您使用 Fize。
FizeCrypt 有非常完善的参考文档，其代码结构也经过精雕细琢，精炼而实用。
